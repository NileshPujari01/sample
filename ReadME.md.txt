This application is minimal Sample API. Currently using postgreSQL as db
Currently it is using
-Clean Architecture
-Automapper
-CQRS
-Entity Framework
-Repository Pattern
-service level logging

It is open to extend with
-inmemory and distributed caching mechanism
-support for Open Telemetry for logging, monitoring, tracing
-Moving credentials in AWS secret and fetch it from there
-authentication and authorisation
-Request/response logging
-API versioning
-Rate limiting
